# Expressions

TF.tf expression data sources build Python expressions and statements that can
be passed to `tensorflow_program`.

See [Basic Example](basic.md) for an end-to-end `local_file` workflow.

Terraform data source type names use the provider prefix:

- `tensorflow_literal`
- `tensorflow_ref`
- `tensorflow_attr`
- `tensorflow_call`
- `tensorflow_assign`

## Literal

`tensorflow_literal` converts JSON into Python literal syntax. Use
`jsonencode(...)` when starting from HCL values.

```hcl
data "tensorflow_literal" "values" {
  value_json = jsonencode([1, 2, 3])
}
```

Output expression:

```python
[1, 2, 3]
```

## Reference

```hcl
data "tensorflow_ref" "float32" {
  name = "tf.float32"
}
```

Output expression:

```python
tf.float32
```

## Call

```hcl
data "tensorflow_call" "constant" {
  function = "tf.constant"
  args     = [data.tensorflow_literal.values.expression]
  kwargs = {
    dtype = data.tensorflow_ref.float32.expression
  }
}
```

Output expression:

```python
tf.constant([1, 2, 3], dtype=tf.float32)
```

## Attribute

```hcl
data "tensorflow_attr" "sequential" {
  receiver = "tf.keras"
  name     = "Sequential"
}
```

Output expression:

```python
tf.keras.Sequential
```

## Assignment

```hcl
data "tensorflow_assign" "x" {
  name  = "x"
  value = data.tensorflow_call.constant.expression
}
```

Output statement:

```python
x = tf.constant([1, 2, 3], dtype=tf.float32)
```

## Program

```hcl
data "tensorflow_program" "main" {
  statements = [
    data.tensorflow_assign.x.statement,
    "print(tf.reduce_sum(${data.tensorflow_assign.x.expression}))",
  ]
}
```

Generated Python:

```python
# Code generated by TF.tf. DO NOT EDIT.

import tensorflow as tf

x = tf.constant([1, 2, 3], dtype=tf.float32)
print(tf.reduce_sum(x))
```
