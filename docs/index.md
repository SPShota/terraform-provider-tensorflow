# TensorFlow Provider

TF.tf generates TensorFlow Python source from Terraform data sources.

The provider exposes generic Python expression builders, generated TensorFlow
API wrappers, `tf.raw_ops` support, and final program assembly through
`tensorflow_program`.

## Example Usage

```terraform
data "tensorflow_literal" "values" {
  value_json = jsonencode([1, 2, 3])
}

data "tensorflow_constant" "x" {
  args = [data.tensorflow_literal.values.expression]
}

data "tensorflow_assign" "x" {
  name  = "x"
  value = data.tensorflow_constant.x.expression
}

data "tensorflow_program" "main" {
  statements = [
    data.tensorflow_assign.x.statement,
    "print(tf.reduce_sum(${data.tensorflow_assign.x.expression}))",
  ]
}
```

Generated Python:

```python
# Code generated by TF.tf. DO NOT EDIT.

import tensorflow as tf

x = tf.constant([1, 2, 3])
print(tf.reduce_sum(x))
```

## Argument Reference

The provider has no configuration arguments.

## Notes

Terraform generates Python source. TensorFlow is required only when running the
generated Python program.
